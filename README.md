# LLM Foundations

A hands-on sprint for building strong foundations in modern LLM engineering and research coding.

## Author

Aravind Shaj  
GitHub: [@aravindshaj](https://github.com/aravindshaj)

## Goal

This repository tracks my work through an LLM foundations sprint, including:

- from-scratch PyTorch implementations
- transformer and attention experiments
- Hugging Face workflows
- LoRA / QLoRA experiments on free GPU runtimes
- paper summaries
- reproducible notes and environment setup

## Repository Structure

- notes/: Learning notes and daily logs
- from_scratch/: PyTorch implementations from scratch
- notebooks/: Colab/Kaggle/Jupyter notebooks
- paper_summaries/: Short summaries of relevant papers
